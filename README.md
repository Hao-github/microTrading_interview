# Kline Project

## Overview

This project converts tick-level CSV data into multiple K-line intervals.

## Structure

- `main.py`: CLI entry point
- `src/kline/`: core package
- `data/input/`: raw input CSV files
- `data/output/`: generated K-line files
- `logs/`: runtime logs
- `checkpoints/`: resume state files
- `tests/`: test placeholders

## Run

```bash
python main.py
```

## Status

Project scaffold only. Business logic is not implemented yet.
